Features

Hear every response. Stay in flow.

Woki Tokie wraps your Claude Code session with ElevenLabs text-to-speech. Each session gets its own voice.

Unique voice per session

Every Claude Code session gets assigned a distinct ElevenLabs voice. Gronk doesn't sound like Blip. You'll know which agent is talking.

Real Claude Code sessions

Not a wrapper. Spawns actual Claude Code CLI processes with full tool use, file editing, and terminal commands. Voice is just the output layer.

Zero config audio

Drop in your ElevenLabs API key and go. Responses stream as audio automatically. No extra setup, no extra windows.

How it works

One command. Your terminal speaks.

Woki Tokie sits between you and Claude Code, catching every response and reading it aloud.

01

Install & run

npx wokitokie launches a wrapped Claude Code session with voice output enabled. Set your ElevenLabs key once.

02

Code normally

Type your prompts like you always do. Woki Tokie intercepts Claude's responses and streams them to ElevenLabs.

03

Listen

Every response gets read aloud in your session's assigned voice. Look at your code, not the terminal. Hear what changed.
